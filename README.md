# 📖 About UniVerse

UniVerse is a **next-generation campus network** where academic collaboration, social interaction, and resource sharing exist in one powerful platform.

Instead of juggling multiple apps for:

- notes
- communication
- communities
- events
- study partners

UniVerse merges them into **one intelligent ecosystem**.

Students discover study partners, faculty share knowledge, communities grow, and campus life becomes digitally connected.

# 🧠 AI Powered Features

UniVerse integrates AI models to enhance safety, networking, and personalized learning experiences.

| Feature | AI Capability |
|---|---|
| 🛡️ NLP Moderation Model | Detect abusive, harmful, or inappropriate text • Flag or filter comments and posts |
| 🖼️ Image Moderation Model | Detect harmful or unsafe images • NSFW classification |
| 🤝 Graph-Based People Recommendation | Suggest students with similar batches • Recommend faculty based on subjects • Friend-of-friend recommendations |
| 📰 Feed Ranking Model | Score posts • Personalize feed based on interests • Engagement and similarity-based ranking |

Powered by OpenAI APIs and intelligent recommendation systems.

<h2>⚙ Installation</h2>

<h3>1️⃣ Clone Repository</h3>

<pre><code>
git clone https://github.com/Salman-Sensei/universe-campus-platform.git
cd universe-campus-platform
</code></pre>

<h3>2️⃣ Install Dependencies</h3>

<pre><code>
npm install
</code></pre>

<h3>3️⃣ Environment Variables</h3>

<p>Create a <code>.env</code> file in the root directory.</p>

<pre><code>
VITE_SUPABASE_URL=
VITE_SUPABASE_PUBLISHABLE_KEY=
OPENAI_API_KEY=
</code></pre>

<h3>4️⃣ Start Development Server</h3>

<pre><code>
npm run dev
</code></pre>

<p>Open the application locally:</p>

<pre><code>
http://localhost:5173
</code></pre>
