

No implementation needed — this is a guidance/research question. The user wants to download their project as a ZIP file.

**Summary:** Lovable does not have a direct "Download ZIP" feature. The standard workflow is to connect the project to GitHub first, then download the ZIP from GitHub's interface.

**Steps:**
1. Go to Project Settings → GitHub → Connect project
2. Authorize and create a repository
3. Visit the GitHub repo and use Code → Download ZIP
